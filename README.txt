An expense tracker app to help you control your spending
This app was built using PySimpleGUI
users can login to the app/ signup as a new user
after the user has been created, the user can first create a budget disclose monthly income and how much they intend to save
at the end of the month user can check the app for insight on their expense 
a dashboard has been included.
